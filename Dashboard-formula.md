Some rules about glossary and formula:

-> Spent => for a task = total of days in the timesheet for the task during the period
         => for a project = total of days in the timesheet in all taks of the project during the period
-> Sold => For a task = field Estimated of the tark
        => for the project = total of estimated of all taks of the project
-> Remaining => for a task = Coloumn Rem. of the task
             => for a project = total of all Rem. of all task of the project
-> Forcast => for a task =  total Spent of the task + Remaining
           => for a project = total Spent of all task of the project + total Rem. of all task of the project.
-> Advancement => for a task = (Forecast of the task / remaining of the task)-1
               => for a  project = (Forcast of the project / Remaining of the project)-1
-> Gap => for a task = (Forecast of the task / sold of the task) -1
       => for a project = (Forecast of the project / sold of the project) -1
